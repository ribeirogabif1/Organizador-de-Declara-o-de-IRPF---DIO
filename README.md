# Organizador-de-Declaracao-IR
Projeto do curso de Excel - SANTANDER + DIO. "CRIAR UM ORGANIZADOR DE DECLARAÇÃO DE IR"

Organizador de Declaração de IRPF Ferramenta inteligente desenvolvida em Excel para centralizar e organizar todos os dados necessários para a Declaração de Imposto de Renda Pessoa Física. O projeto foca em experiência do usuário (UX), utilizando menus de navegação, validações de dados e automação para facilitar o preenchimento e evitar erros comuns no processo de declaração.
